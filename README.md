# liri-node-app
This is an app created in NodeJs with following packages used:
* dotenv
* axios
* moment
* inquirer
* spotify-node-api
### APIs:
* Spotify
* OMBD
* BandsInTown
### How to run
* Select any from following menu and you will get the data about future concerts, favorite song and movie.
